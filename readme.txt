Youtube Like Thumbs Up Or Thumbs Down System For Voting On Posts And Comments.

== Description ==
Like(thumbs up) or Dislike(thumbs down) Voting plugin for each posts & comments. Through the admin panel, you can configure the plugin to restrict voting system only for registered user only or guest both. 

== Installation ==
1. Upload `YoutubeLikeRating` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Wp-admin -> Setting -> YoutubeLikeRating.
4. Update setting as per your requirement.
5. Check on the front-end & enjoy voting!!!.


== Frequently asked questions ==
Why this Plugin?

Like(thumbs up) or Dislike(thumbs down) Voting plugin for each posts & comments. Through the admin panel, you can configure the plugin to restrict voting system only for registered user only or guest both. 


Can I configure the plugin to restrict only on post only or comment only?

Yes, you can do this setting from admin panel, admin panel has dropdown called "Allow vote on" with options "Post only", "Comment only" and "Both Post and comment" so you can choose one option as per your requirement.


Can I configure the plugin to restrict voting system to registred user only?

Yes, you can set that option from admin panel.
